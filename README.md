# DIR

This repository is an official implementation of the ICCV 2023 paper "Decoupled Iterative Refinement Framework for Interacting Hands Reconstruction from a Single RGB Image".

> ####  [Decoupled Iterative Refinement Framework for Interacting Hands Reconstruction from a Single RGB Image](https://arxiv.org/abs/2302.02410)
> ##### [Pengfei Ren](https://pengfeiren96.github.io/), [Chao Wen](https://scholar.google.com/citations?user=v8TFZI4AAAAJ), [Xiaozheng Zheng](https://scholar.google.com/citations?user=3hSD41oAAAAJ), [Zhou Xue](https://scholar.google.com/citations?user=ECKq3aUAAAAJ), [Haifeng Sun](https://scholar.google.com/citations?user=dwhbTsEAAAAJ), [Qi Qi](https://scholar.google.com/citations?user=2W2h0SwAAAAJ), [Jingyu Wang](https://jericwang.github.io/), [Jianxin Liao](https://www.researchgate.net/scientific-contributions/Jianxin-Liao-8024422)
